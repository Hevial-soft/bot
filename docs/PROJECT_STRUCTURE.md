# Hevial-bot - Структура проекта

## 📁 Структура директорий

```
hevial-bot/
├── src/                           # Исходный код приложения
│   ├── bot/                       # Логика Telegram бота
│   │   ├── index.js              # Главный файл бота
│   │   └── handlers/             # Обработчики команд
│   │       ├── dialog.js         # Диалоги с пользователем
│   │       ├── notification.js   # Уведомления
│   │       └── pricing.js        # Управление ценами
│   ├── services/                 # Бизнес-логика
│   │   ├── ai.js                 # Интеграция с AI (Claude)
│   │   └── scheduler.js          # Запланированные задачи
│   ├── db/                       # Работа с базой данных
│   │   └── index.js              # Подключение к БД
│   ├── middleware/               # Middleware для обработки
│   │   └── session.js            # Управление сессиями
│   ├── config/                   # Конфигурация
│   │   └── config.js             # Загрузка переменных окружения
│   └── index.js                  # Точка входа приложения
├── database/                      # SQL и схемы БД
│   └── schema.sql               # Схема базы данных PostgreSQL
├── tests/                        # Тесты
├── docs/                         # Документация
├── .env.example                  # Пример переменных окружения
├── .gitignore                    # Исключения для git
├── package.json                  # Зависимости и скрипты
├── README.md                     # Документация проекта
└── index.js                      # Главная точка входа

```

## 🚀 Быстрый старт

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Создайте .env файл:**
   ```bash
   cp .env.example .env
   ```
   Отредактируйте `.env` с вашими учетными данными

3. **Запустите в режиме разработки:**
   ```bash
   npm run dev
   ```

4. **Запустите в продакшене:**
   ```bash
   npm start
   ```

## 📦 Зависимости

- **telegraf** - Telegram Bot API
- **pg** - PostgreSQL драйвер
- **@anthropic-ai/sdk** - AI интеграция (Claude)
- **dotenv** - Загрузка переменных окружения
- **nodemon** - Hot-reload для разработки

## 🗄️ База данных

- **Тип:** PostgreSQL
- **Схема:** [database/schema.sql](database/schema.sql)

### Требуемые переменные окружения (в `.env`):
- `BOT_TOKEN` - Токен Telegram бота от @BotFather
- `DB_HOST` - Хост БД (по умолчанию: localhost)
- `DB_PORT` - Порт БД (по умолчанию: 5432)
- `DB_USER` - Пользователь БД (по умолчанию: postgres)
- `DB_PASSWORD` - Пароль БД
- `DB_NAME` - Имя БД (по умолчанию: hevial_db)
- `ANTHROPIC_API_KEY` - API ключ для Claude AI

## 📝 Структура модулей

### src/bot/
Главная логика Telegram бота. Точка старта всех взаимодействий с пользователем.

### src/services/
Бизнес-логика, отделенная от ботовскую:
- AI сервис для обработки текста
- Scheduler для автоматизированных задач

### src/db/
Подключение и управление базой данных PostgreSQL.

### src/middleware/
Middleware компоненты для обработки:
- Управление сессиями пользователей

### src/config/
Централизованное управление конфигурацией приложения.

## ✅ Лучшие практики

- Каждый модуль отвечает за одну задачу
- Бизнес-логика отделена от обработчиков бота
- Конфигурация централизована
- Все зависимости указаны в package.json
